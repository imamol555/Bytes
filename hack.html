What is password cracking?


It is basically the process of recovering passwords to breach (or restore) the security of a computer system. Password cracking is an integral part of digital forensics and pretesting. The purpose of password cracking revolves around recovering the forgotten passwords of our online accounts, computers, and smartphones.


Talking about the process of password cracking, most methods involve the use of a computer that generates a vast set of password candidates. A desktop computer tests more than hundreds of millions of passwords per second.


The time needed to crack a password is proportional to the length and strength of that password. That’s why users are advised to use complex passwords that are harder to guess.


Types of password cracking attacks:


  • 1.Dictionary attack: This attack uses a file that contains a list of words that are found in the dictionary.
  • 2.Brute force attack: Apart from the dictionary words, brute force attack makes use of non-dictionary words too.
  • 3.Rainbow table attack: This attack comes along with pre-computed hashes. Thus, this method is faster.


There are lots of other password cracking techniques like phishing, spidering, social engineering, shoulder surfing etc.

LOphtcrack


20 years ago, L0phtcrack was the first password auditing tool for Microsoft Windows operating system. Back then, L0phtCrack was able to crack the 8-character Windows password in about 24 hours. L0phtcrack 7 has been released recently which has come after seven years.

Apart from faster password cracking, L0phtCrack 7 comes with a better password auditing wizard, reporting, and scheduling. The new release works with all versions of Windows along with the support for many new UNIX password hashes.
